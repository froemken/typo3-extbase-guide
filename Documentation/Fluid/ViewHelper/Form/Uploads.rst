f:form.uploads
==============

Mit diesem ViewHelper erstellst Du ein Uploadfeld, um Dateien an den Server zu senden.

Eigenschaften
-------------

Dieser ViewHelper besitzt keine Eigenschaften

Beispiel
--------

::

 <f:form.upload name="myExtName[image]" />

oder

::

 <f:form.upload property="image" />
