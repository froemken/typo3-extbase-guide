Globale Eigenschaften für das HTML-Element
##########################################

.. t3-field-list-table::
 :header-rows: 1

 - :Property,20:    Eigenschaft
   :Datatype,20:    Datentyp
   :Description,40: Beschreibung
   :Standard,10:    Standard
   :Mandatory,10:   Mandatory

 - :Property:    class
   :Datatype:    String
   :Description: CSS-Klasse für dieses HTML-Element
   :Standard:    NULL
   :Mandatory:   Nein

 - :Property:    dir
   :Datatype:    String
   :Description: Die Textrichtung. Erlaubte Werte sind: ltr (links nach rechts) und rtl (rechts nach links)
   :Standard:    NULL
   :Mandatory:   Nein

 - :Property:    id
   :Datatype:    String
   :Description: Ein Bezeichner, der in dem Dokument ein HTML-Element eindeutig identifiziert
   :Standard:    NULL
   :Mandatory:   Nein

 - :Property:    lang
   :Datatype:    String
   :Description: Angabe der Sprache nach RFC 1766, die in diesem HTML-Element verwendet wird
   :Standard:    NULL
   :Mandatory:   Nein

 - :Property:    style
   :Datatype:    String
   :Description: Individuelle CSS-Stile, die nur für dieses HTML-Element gültig sein sollen
   :Standard:    NULL
   :Mandatory:   Nein

 - :Property:    title
   :Datatype:    String
   :Description: Ein Text, der beim berühren mit der Maus als Tooltip erscheinen soll
   :Standard:    NULL
   :Mandatory:   Nein

 - :Property:    accesskey
   :Datatype:    String
   :Description: Eine Tastenkombination, mit der dieses HTML-Element direkt angesprungen werden kann
   :Standard:    NULL
   :Mandatory:   Nein

 - :Property:    tabindex
   :Datatype:    Integer
   :Description: Gibt die TAB-Reihenfolge an
   :Standard:    NULL
   :Mandatory:   Nein

 - :Property:    onclick
   :Datatype:    String
   :Description: JavaScript Code, der bei Klick auf dieses HTML-Elemtent ausgeführt werden soll
   :Standard:    NULL
   :Mandatory:   Nein
